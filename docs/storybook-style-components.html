Quick Start Guide

Integrate UX4G Flutter components quickly with the same structure as your UX4G Storybook docs: setup, what is included, and component API details.

Install via pubspec

dependencies:
  ux4g_flutter_design_system:
    path: ../ux4g-flutter-design-system

Wrap app with theme

MaterialApp(
  theme: Ux4gTheme.themeData(isDark: false),
  darkTheme: Ux4gTheme.themeData(isDark: true),
)

Example usage

Ux4gButton(
  onPressed: () {},
  text: 'Get Started',
)

What is included

  • Foundations: colors, typography, spacing, icons
  • Components: forms, feedback, navigation, data display
